## Forensic Methodology: How We Built This Analysis

**The 47-Contract Calculation**

Sierra's public data:
- $100M ARR confirmed
- 31 named enterprise customers
- Claims to serve >95% of Black Friday shoppers, >50% of families in healthcare, >70% of fintech value chain

**The Math:**
```
Known: 31 customers = $77M ARR (from detailed analysis above)
Remaining: $100M - $77M = $23M ARR
Missing contracts: $23M รท $1.4M avg = 16 additional contracts
Total: 31 + 16 = 47 contracts
```

## The Risk

**Performance dependency**: Traditional SaaS fails, customers complain. Sierra fails, customers stop paying immediately.

If Rocket Mortgage's 4x conversion drops to 2x, the contract value drops proportionally. This isn't seat-based licensing where usage stays predictable.

**Scale risk**: Adding customers could degrade existing performance. There's no clear path to scale without performance trade-offs.

## What We Learned

1. **AI agents can generate $2M+ ACVs** when tied to revenue outcomes
2. **Enterprise adoption is real** - 47 Fortune 1000 companies proved it
3. **The pricing model is fragile** - performance = revenue in real time

Sierra built the template. Now they need to prove it scales.
